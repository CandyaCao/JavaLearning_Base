## 文件
File类：是管理文件或目录的方法。是文件或目录路径名的抽象表达形式

unix：严格区分大小写
windows：不区分大小写

unix：“/” 使用“/”

URI：统一资源标识符，
URL：统一资源定位符
	URL   协议：//IP地址：端口号/资源路径




## 操作步骤
1. 创建源或目标文件
	输入：把文件中的数据输入到程序中，文件就是源
	输出：把程序中产生的数据输出到文件中，文件就是目标
2. 创建I/O流对象
	输入操作：创建输入流
	输出操作：创建输出流
3. 进行具体的IO操作
	输入：read
	输出：write
4. 关闭资源
	所有的流对象都有close方法，用于关闭流资源

### 字符流与字节流
1. 字符流效率肯定比字节流高，字节流读中文就可能会出现乱码。一般来说，文本文件用字符流读，但是二进制文件(音频、视频、图片)只能用字节流读
